A modern, responsive personal portfolio website showcasing web development projects, skills, and blog posts.

## Site Map

- `/portfolio-index.html` - Home page
- `/about.html` - About me page with education, skills, and experience
- `/projects.html` - Portfolio projects showcase
- `/gallery.html` - Photo gallery of design work
- `/blog.html` - Blog listing page
- `/post-1.html` - Sample blog post
- `/contact.html` - Contact form page
- `/map.html` - Location map and video content
- `/policy.html` - Privacy policy and terms of use

## Features

### HTML & Structure
-  8 separate HTML files
-  Semantic HTML5 tags (header, nav, main, section, article, footer)
-  Consistent navigation bar across all pages
-  Internal and external links
-  Multiple images with alt text
-  Figure/figcaption elements
-  Iframe integration (Google Maps, YouTube)

### CSS & Styling
-  External CSS file (`assets/css/styles.css`)
-  Flexbox and Grid layouts
-  Responsive design with breakpoint at 768px
-  CSS variables for colors
-  Hover and focus states
-  Consistent color scheme and typography
-  Print-friendly styles

### Optimization & Quality
-  Optimized image structure (ready for WebP/JPG)
-  Favicon support
-  Meta viewport tag
-  Meta description tags
-  Unique title tags for each page
-  Clean folder structure
-  Semantic HTML ready for W3C validation

## How to Run

1. Clone or download this repository
2. Open `portfolio-index.html` in a web browser
3. Navigate through the pages using the navigation menu

## Deployment

This website can be deployed to:
- **GitHub Pages**: Push to a GitHub repository and enable Pages
- **Netlify**: Drag and drop the folder or connect via Git
